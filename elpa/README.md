# Prefered Packages

Maintaining ELPA packages via source control usually ends up in a world of hurt. Instead here's a list of useful packages to be installed by hand via EPLA.

* rhtml-mode
* magit
* multiple-cursors
* markdown-mode
* git-gutter+
* rainbow-delimiters
* yaml-mode
* zenburn-theme
* multiple-cursor
* multi-term
* web-mode
